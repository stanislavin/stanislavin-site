# 🎮 Olga the Designer - 2D Platformer

A birthday gift game! Help Olga save money for her dream apartment by defeating enemies across three cities.

## 🎯 Quick Start

Simply open `index.html` in your web browser and click "Начать игру"!

## 🕹️ Controls

- **Movement**: ← → / A D
- **Jump**: ↑ / W / SPACE
- **Attack**: F (with shooter power-up)

## 🌍 Levels

1. **Moscow** - Where it all began
2. **Belgrade** - Fancy joins you!
3. **St. Petersburg** - Face the final boss Ganvest

## 💰 Objective

- Defeat enemies to earn rubles
- Collect power-ups for special abilities
- Reach the 🏠 at the end of each level
- Save enough money to buy your dream apartment!

## 🎨 Features

- Beautiful pixel art graphics
- 3 unique levels with different backgrounds
- Enemy AI and boss fights
- Power-up system
- Fancy companion character
- Boss says "фа пепе шнейне"!

## 🏆 Victory

Complete all three levels to see Olga finally get her apartment!
